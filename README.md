# Set-TeamsCache.ps1 #
### Written by Anthony Cesario ###

## Purpose ##

This script exists to automate the process of resetting Teams cache.

## Audience ##

This script was written for tier one / two techs who find themselves resetting Teams cache often and may want to speed up the process.

## Prerequisites ##

There are no prerequisites to run this script.

## Procedure ##

  1. Copy down the script
  2. Open up a PowerShell window and cd to the directory where the script was downloaded to.
  3. Run the command ".\Set-TeamsCache.ps1" and the script will run.
        3a. Teams will close and the entire MSTeams_8wekyb3d8bbwe directory will be deleted. Teams will then be opened back up.
  5. You will be presented with a GUI. Press accept for the rest of the script to run, press deny to close the window and end the script there.
